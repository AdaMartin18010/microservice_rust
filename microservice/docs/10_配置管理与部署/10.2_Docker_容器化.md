# 10.2 Docker 容器化

## 10.2.1 多阶段构建

```Dockerfile
FROM rust:1.90 as builder
WORKDIR /app
COPY . .
RUN cargo build --release

FROM gcr.io/distroless/cc
COPY --from=builder /app/target/release/microservice-server /usr/bin/app
USER 65532:65532
ENTRYPOINT ["/usr/bin/app"]
```

## 10.2.2 最佳实践

- 最小基础镜像、只读根文件系统；
- 显式暴露端口与健康检查；
- SBOM 与镜像签名（cosign）。
